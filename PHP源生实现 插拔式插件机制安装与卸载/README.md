## 文件介绍

Index.php       //插件列表

Add.php         //添加新插件

AddInfo.php     //添加新插件处理

Del.php         //删除插件

Info.php        //插件安装,卸载,禁用,开启处理

Preview.php     //插件功能预览

Hooks.class.php //插件机制基类


## 目录介绍

Public          //样式文件目录

plugins         //插件文件存放目录，该目录下的每一个文件夹都是一个插件
